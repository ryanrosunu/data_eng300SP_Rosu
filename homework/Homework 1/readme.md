## How to Run
1. Open the Jupyter notebook `hw1.ipynb`
2. Ensure the following required libraries are installed in local python environment:
- pandas
- numpy
- missingno
- seaborn
- matplotlib
- scipy
- sklearn
3. Run the code in order

## Expected Output
- Problem 1: Several print statements, tables, and plots useful for missingness exploration. Also, several imputations on different columns of the original dataframe.
- Problem 2: Several print statements including long lists used for data exploration. Several standardizations applied to different columns of the orignal dataframe.
- Problem 3: The removal of rows from the original dataframe with any missing data.
- Problem 4: Several print statements and plots useful for exploration of distributions. Also, the addition of two new data columns with box-cox transformations applied.
- Problem 5: Several print statements and plots useful for exploration of proportions. Also, the addition of a new data column for describing the general size class of an aircraft.
- Problem 6: A table describing all the numeric columns used and several print statements displaying the results of different machine learning model's RMSE outcomes.
